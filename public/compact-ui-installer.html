FoodBao Ultra-Compact UI Installer

This tool allows you to instantly apply the Ultra-Compact UI to any FoodBao application page. Simply drag the bookmarklet below to your bookmarks bar, then click it when viewing any FoodBao page.

How to use:

  1. Drag the bookmarklet below to your browser's bookmarks bar
  2. Navigate to any FoodBao application page

Features of Ultra-Compact UI

  • Reduces spacing throughout the entire application
  • Makes optimal use of screen real estate
  • Perfect for tablets, small screens, and kitchen displays
  • Reduces header height and button sizes
  • Compacts lists, tables, and cards
  • Optimizes modals and form elements
  • Works instantly without page reloads

For Developers

You can also activate Ultra-Compact UI by running the following code in your browser's developer console:

const script = document.createElement('script'); script.src = '/js/compact-ui-installer.js?v=' + new Date().getTime(); document.head.appendChild(script);

Permanent Installation

To permanently enable Ultra-Compact UI for all pages, include this line in the <head> section of your HTML:

<script src="/js/global-compact-ui-loader.js"></script>
